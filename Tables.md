## You can create tables using the | and - symbols.
### For Example :
```
| Header 1 | Header 2 |
|----------|----------|
| Row 1, Cell 1 | Row 1, Cell 2 |
| Row 2, Cell 1 | Row 2, Cell 2 |
```
### Outputs:
| Header 1 | Header 2 |
|----------|----------|
| Row 1, Cell 1 | Row 1, Cell 2 |
| Row 2, Cell 1 | Row 2, Cell 2 |
