## You can create a task list with checkboxes using - [ ] for an unchecked box, and - [x] for a checked box.
### For Example: 
```
- [x] Task 1 (Completed)
- [ ] Task 2 (Not Completed)
- [ ] Task 3 (Pending)
```
### Outputs:
- [x] Task 1 (Completed)
- [ ] Task 2 (Not Completed)
- [ ] Task 3 (Pending)
