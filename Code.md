## Inline Code: Use single backticks (`) for inline code.
### For Example:
```
`
This single backticks for inline code.
`
```
### Outputs

` This is Inline Code`

## Code Block: Use triple backticks (```) for code blocks.
```
```This triple backticks for  code block.```
```
### Outputs
 ```
This triple backticks for  code block.
```
